Project Detail / 프로젝트 상세

mal_project

악성코드가 어떻게 동작하고 어떻게 탐지할 수 있는지를 직접 실습해보기 위해 진행한 분석 프로젝트.

GitHub ↗ 프로젝트 목록으로

Overview / 개요

이 프로젝트는 악성코드가 실제로 어떤 방식으로 동작하는지 이해하기 위해 시작한 실습형 분석 프로젝트다. 특히 최근 악성코드가 운영체제나 프로그램 안에 이미 존재하는 합법적인 도구와 기능을 활용하는 경우가 많다는 점에 관심을 두고, 그 흐름을 직접 확인해보고 싶어서 진행하게 됐다.

즉 단순히 자료를 읽고 정리하는 데서 그치지 않고, 동작 원리와 탐지 흐름을 직접 다뤄보면서 학습 내용을 프로젝트 형태로 남기는 것이 핵심이었다.

Project Diagram / 구성도

분석, 행위 관찰, 탐지 규칙 설계, YARA 매칭 흐름을 요약한 악성코드 분석 구성도.

Why I Built It / 만든 이유

악성코드가 단순히 독립적인 실행 파일 하나로만 존재하는 것이 아니라, 합법적인 시스템 도구나 애플리케이션 내부 기능을 활용해 동작하는 경우가 있다는 점이 흥미로웠다. 그래서 그런 흐름을 단순히 이론으로만 이해하는 것이 아니라, 직접 실습해보면서 동작 원리와 분석 포인트를 체감해보고 싶었다.

My Role / 역할

이 프로젝트에서는 엑셀 매크로에 사용되는 VBA 코드와 악성코드 동작 요소를 직접 다뤘다. 즉 단순히 개념만 정리한 것이 아니라, 동작을 구성하고 실습하는 쪽에 직접 손을 댄 프로젝트였다.

How I Want It Framed / 어떻게 소개하고 싶은가

이 프로젝트는 “악성코드 분석 실습 프로젝트”로 소개하는 것이 가장 적절하다. 핵심은 위험한 행위를 과장하는 데 있는 것이 아니라, 동작 원리와 탐지 방법을 직접 이해하기 위해 실습 중심으로 접근했다는 점에 있다.

What Stands Out / 강조할 포인트

  • 단순 이론 학습이 아니라 직접 실습해본 프로젝트
  • 악성코드 동작 분석과 구현 흐름을 함께 다룸
  • YARA 룰을 만들어 탐지까지 함께 다룸
  • 분석, 실습, 탐지라는 세 흐름을 하나의 프로젝트로 연결
