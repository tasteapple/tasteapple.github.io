Project Detail / 프로젝트 상세

FrontScope

AI를 활용해 프론트엔드 표면 분석용 도구를 직접 만들어본 프로젝트.

GitHub ↗ 프로젝트 목록으로

Overview / 개요

FrontScope는 단순히 프론트엔드 표면을 분석하는 도구를 만든 프로젝트이기도 하지만, 동시에 AI를 활용해 실제 보안 도구를 어느 정도까지 설계하고 확장할 수 있는지를 실험해본 프로젝트이기도 하다.

이 프로젝트의 출발점은 “AI만 사용해서 표면 분석용 툴을 만들어 보면 어디까지 갈 수 있을까?”라는 질문이었다. 그 과정에서 브라우저 표면 분석, 자산 수집, 크롤링, 리포트 생성 같은 요소들을 하나의 흐름으로 묶어보는 방향으로 발전했다.

Core Focus / 핵심 방향

  • 프론트엔드 및 브라우저 표면 분석
  • 자산, 스크립트, 노출 요소 수집
  • 크롤링 기반 확장 탐색
  • 리포트 생성과 결과 정리

Why I Built It / 만든 이유

최근에는 AI를 활용해 제품이나 도구를 만드는 흐름이 점점 많아지고 있다. FrontScope는 그 흐름 속에서 “직접 AI를 사용해 실질적인 분석 도구를 만들어보자”는 목적에서 시작됐다.

즉 이 프로젝트는 특정 기능 하나를 구현하는 데 그치지 않고, AI를 통해 아이디어를 구조화하고 기능을 붙여가며 실제 결과물까지 만들어보는 경험 자체에 의미가 있었다.

My Role / 역할

이 프로젝트에서 나는 직접 모든 세부 구현을 손으로 쌓아 올렸다기보다, AI에게 어떤 기능을 추가할지, 어떤 방향으로 발전시킬지, 무엇을 더 분석하게 만들지 지시하면서 전체 흐름을 주도했다.

즉 이 프로젝트는 AI를 활용한 기획·지시·조정의 결과물이라는 성격이 강하다.

How I Want It Framed / 어떻게 소개하고 싶은가

FrontScope는 “AI 활용도를 높이기 위해 실제로 만들어본 보안 도구 프로젝트”로 소개하는 것이 가장 잘 맞는다. 이 프로젝트의 핵심은 완성된 제품 하나보다, AI를 사용해 도구를 빠르게 설계하고 확장하는 과정을 직접 경험했다는 점에 있다.

What Stands Out / 강조할 포인트

  • AI를 사용한 실전형 보안 도구 제작 실험
  • 프론트엔드 표면 분석, 자산 수집, 크롤링, 리포팅을 하나의 흐름으로 연결
  • 단순 아이디어 수준이 아니라 실제로 공개 가능한 결과물로 정리
  • AI를 활용한 개발 워크플로우를 포트폴리오 자산으로 남긴 프로젝트

Project Diagram / 구성도

FrontScope가 타깃 URL에서 시작해 크롤링, 자산 수집, 프론트엔드 분석, 그리고 리포트 생성까지 이어지는 흐름을 정리한 구성도.
