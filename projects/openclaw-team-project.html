Project Detail / 프로젝트 상세

OpenClaw Team Project

OpenClaw 기반 AI Agent 환경을 직접 구성하고, 공격/방어 관점에서 테스트와 산출물 정리까지 진행한 팀 프로젝트.

프로젝트 목록으로

Overview / 개요

이 프로젝트는 OpenClaw를 직접 써보고 싶다는 호기심에서 시작됐다. 요즘 AI agent가 빠르게 확산되고 있고, 앞으로는 온디바이스 환경에도 AI와 agent 활용이 자연스럽게 들어올 것이라고 생각했기 때문에, 실제 환경을 직접 구성하고 검토해보고 싶었다.

프로젝트는 이후 단순 사용 경험을 넘어서, OpenClaw 기반 AI Agent 환경에서 어떤 취약점이 있는지, 어떤 공격이 통하는지, 그리고 버전이나 모델에 따라 결과가 어떻게 달라지는지를 확인하는 공격/방어 테스트 흐름으로 발전했다.

Core Focus / 핵심 내용

  • OpenClaw 직접 구성 및 사용
  • 취약점 및 공격/방어 테스트
  • RCE와 프롬프트 인젝션 시도
  • 모델 버전별 동작 차이 확인
  • 최종 발표 자료 및 결과물 정리

Why I Built It / 만든 이유

가장 큰 이유는 OpenClaw를 직접 써보고 싶었기 때문이다. 요즘 AI agent가 점점 더 주목받고 있고, 결국에는 온디바이스 환경에도 AI들이 탑재되고 agent 활용이 일상화될 것이라고 생각했다. 그래서 실제 환경을 직접 구성하고, 동작 방식과 보안 측면을 스스로 확인해보고 싶었다.

My Role / 역할

이 프로젝트에서 나는 기획, OpenClaw 구성, 그리고 취약점 테스트를 맡았다. 단순 사용이 아니라 직접 환경을 만들고, 테스트 시나리오를 잡고, 어떤 부분이 동작하고 어떤 부분이 막히는지 검토하는 쪽에 집중했다.

How I Want It Framed / 어떻게 소개하고 싶은가

이 프로젝트는 ‘OpenClaw 기반 팀 프로젝트’이자 ‘AI Agent 공격/방어 테스트 프로젝트’로 소개하는 것이 가장 잘 맞는다. 핵심은 단순히 OpenClaw를 사용해본 것이 아니라, 실제로 구성하고, 취약점 관점에서 테스트하고, 그 결과를 팀 산출물과 발표 자료로 정리했다는 점에 있다.

What Stands Out / 강조할 포인트

  • OpenClaw를 직접 구성하고 실제로 사용해본 프로젝트
  • RCE, 프롬프트 인젝션 등 공격/방어 테스트를 수행
  • OpenClaw 버전과 모델 버전에 따라 동작 차이를 관찰
  • 최종 발표 자료와 팀 산출물까지 정리한 프로젝트

Visual Notes / 시각 자료 메모
