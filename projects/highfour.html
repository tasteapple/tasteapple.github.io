HighFour

Overview / 개요

HighFour는 파이썬, 머신러닝, 딥러닝을 배우고 난 뒤, 그 내용을 실제로 써먹어 보기 위해 진행한 팀 프로젝트다. 프로젝트의 핵심 아이디어는 사용자가 증상을 입력하면 가능한 병명을 유추해주고, 필요할 경우 근처 병원을 추천해주는 프로그램을 만드는 것이었다.

즉 단순한 학습 정리에서 끝나는 것이 아니라, 배운 기술을 실제 동작하는 서비스 형태로 연결해보는 데 의미가 있었다.

Core Focus / 핵심 내용

  • 증상 기반 병명 유추
  • 근처 병원 추천
  • 실제 동작 시연
  • 머신러닝/딥러닝 모델 적용

Why I Built It / 만든 이유

이 프로젝트를 만든 이유는 파이썬, 머신러닝, 딥러닝을 배우고 끝내는 것이 아니라, 실제로 쓸 수 있는 형태로 한 번 만들어보고 싶었기 때문이다. 즉 배운 내용을 실제 서비스 형태에 가깝게 적용해보는 것이 핵심 동기였다.

My Role / 역할

이 프로젝트에서 나는 모델을 직접 만들고, 여러 모델을 실험하고 비교하면서 가장 나은 결과를 내는 방향을 찾는 역할을 맡았다. 즉 단순히 팀 프로젝트에 참여한 것이 아니라, 핵심이 되는 모델링과 트레이닝을 직접 담당했다.

How I Want It Framed / 어떻게 소개하고 싶은가

이 프로젝트는 ‘협업형 팀 프로젝트’로 소개하는 것이 가장 잘 맞는다. 동시에, 배운 기술을 실제 구현으로 연결하고, 모델을 직접 만들고 학습시켜 동작하는 결과물까지 만들어낸 프로젝트라는 점도 함께 보여주고 싶다.

What Stands Out / 강조할 포인트

  • 배운 파이썬, 머신러닝, 딥러닝을 실제 프로젝트에 적용
  • 직접 모델을 만들고 트레이닝함
  • 실제 동작 시연이 가능한 형태로 구현
  • 협업 구조, 결과물, 발표 흐름까지 포함된 팀 프로젝트
