Projects / 프로젝트

보안, AI, 그리고 실제 구현 경험을 담은 프로젝트들

이 페이지에는 내가 직접 만들거나 팀과 함께 진행한 프로젝트들을 정리했다. 프론트엔드·백엔드 보안 분석 도구, 악성코드 분석 실습, LLM 보안 진단, AI agent 공격/방어 테스트, 그리고 머신러닝·블록체인 기반 팀 프로젝트까지, 관심사와 실전 경험이 연결되는 작업들을 모아두었다.

FrontScope 브라우저와 프론트엔드 표면을 분석하고 리포트로 정리하는 보안 정찰 프로젝트. BackScope 라우트, 파라미터, 응답 행위를 중심으로 백엔드 표면을 분석하는 프로젝트. mal_project 악성코드 분석과 구현 실습 흐름을 담고 있는 프로젝트. HighFour 증상 기반 병명 유추와 병원 추천을 위해 머신러닝 모델을 직접 만들고 학습시킨 협업형 프로젝트. voice_pishing_team_project 보이스피싱이라는 사회 문제를 배경으로 블록체인을 활용한 서비스/앱 구현을 시도한 협업형 프로젝트. OpenClaw Team Project OpenClaw 기반 AI Agent 환경을 직접 구성하고 공격/방어 테스트까지 수행한 팀 프로젝트. LLM 취약점 진단 리스트 작성 및 테스트 LLM 보안 점검 항목을 정리하고 테스트 흐름으로 연결하는 프로젝트.
