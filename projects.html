Projects / 프로젝트

학습, 실험, 구현 결과가 어떻게 프로젝트로 이어졌는지 보여주는 페이지

각 카드는 별도의 프로젝트 상세 페이지로 연결된다. 먼저 프로젝트의 맥락과 의미를 보고, 이후 필요하면 GitHub로 이동하는 흐름을 의도했다.

FrontScope 프론트엔드 및 브라우저 표면을 분석하고, 자산·크롤링·리포트를 다루는 보안 정찰 프로젝트. BackScope 백엔드 표면, 라우트, 파라미터, 응답 행위를 수집·비교·정리하는 보안 분석 프로젝트. mal_project 악성코드 분석과 구현 실습 흐름을 담고 있는 프로젝트. HighFour 기획, 역할 분담, 개발 흐름, 팀 실행 과정을 보여주는 협업형 프로젝트. voice_pishing_team_project 시나리오 작성, 구현, 발표 준비까지 이어지는 팀 프로젝트형 작업. OpenClaw Team Project OpenClaw 구성, 기능 정의, 구현, 산출물 정리 흐름이 보이는 팀 프로젝트. 악성코드 제작 및 탐지 악성코드 제작과 탐지라는 주제를 실습과 시연 중심으로 다루는 프로젝트. LLM 취약점 진단 리스트 작성 및 테스트 LLM 보안 점검 항목을 구조화하고 테스트 흐름으로 연결하는 프로젝트.
