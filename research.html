Research / 연구

보안과 AI가 만나는 지점을 계속 파고드는 중이다

웹 보안, 악성코드 분석, LLM 보안, AI agent 보안처럼 직접 손을 대보면서 감을 잡아야 하는 주제들에 특히 관심이 많다. 이 페이지는 그런 관심사와 실습 흐름, 그리고 앞으로 더 깊게 가져가고 싶은 방향을 정리해두는 공간이다.

Focus Areas / 관심 분야

Web Security Malware Analysis LLM Security AI Agent Security Security Automation

최근에는 프론트엔드와 백엔드 보안 분석 도구를 직접 만들고, LLM 보안 진단 항목을 정리하고, OpenClaw 같은 AI agent 환경을 직접 구성해 보안 관점에서 테스트해보는 쪽으로 흐름이 이어지고 있다.

Research Style / 방식

  • 실습과 구현 기반으로 배우는 편
  • 체크리스트와 정리를 중요하게 보는 편
  • 도구를 직접 만들며 이해하는 방식 선호
  • 결과를 문서화하고 다시 재사용 가능한 형태로 남기려는 편

Current Themes / 지금 주로 보는 것

  • 브라우저와 프론트엔드 표면 분석
  • 라우트, 파라미터, 응답 행위 기반 백엔드 분석
  • Prompt Injection, Jailbreak 등 LLM 취약점 분류와 테스트
  • AI agent 환경에서의 권한, 도구 사용, 프롬프트 오염 문제

How It Connects / 어떻게 이어지는지

지금 다루는 주제들은 따로 노는 것처럼 보여도 결국 서로 연결되어 있다. 웹 보안과 악성코드 분석은 기본적인 공격·방어 감각을 키워주고, LLM 보안과 AI agent 보안은 앞으로 더 중요해질 환경에서 그 감각을 다시 적용해보게 만든다. 그래서 최근에는 전통적인 보안 주제와 AI 보안 주제를 같이 가져가고 있다.

Research Direction / 앞으로 더 보고 싶은 것

앞으로는 웹과 API 보안 분석 경험을 더 쌓으면서, LLM 보안 진단 체계와 AI agent 보안 테스트를 더 정교하게 가져가고 싶다. 단순히 취약점을 나열하는 수준이 아니라, 실제로 점검 항목을 만들고 자동화 도구로 연결하고, 테스트 결과를 다시 재사용 가능한 형태로 남기는 흐름까지 만드는 것이 목표다.
